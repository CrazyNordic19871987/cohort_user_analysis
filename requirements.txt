Этот файл requirements.txt содержит все необходимые зависимости для вашего проекта когортного анализа. Вы можете установить их все сразу с помощью команды:

pip install -r requirements.txt

Основные библиотеки и их назначение:

pandas - для работы с данными и анализа

numpy - для числовых операций

matplotlib - для базовой визуализации

seaborn - для улучшенных визуализаций

openpyxl - для чтения Excel файлов

jupyter - для работы с Jupyter notebook

Файл использует минимальные версии, которые совместимы с кодом проекта.
